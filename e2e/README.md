# E2E Test Suite

## Quick Start

```bash
# Install browsers (first time only)
npx playwright install

# Run all tests
npm test

# Run tests with UI
npm run test:ui

# View test report
npm run test:report
```

## Test Files

- `homepage.spec.ts` - Homepage functionality
- `navigation.spec.ts` - Page navigation
- `deals.spec.ts` - Deal browsing and interaction
- `search.spec.ts` - Search functionality
- `subscription.spec.ts` - Email subscription
- `responsive.spec.ts` - Responsive design
- `performance.spec.ts` - Performance and SEO
- `accessibility.spec.ts` - Accessibility compliance
